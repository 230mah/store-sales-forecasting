# Store Sales Forecasting

Este projeto realiza uma análise e previsão das vendas em lojas utilizando técnicas de machine learning.

## Estrutura do projeto

- `data/raw/` - Dados originais
- `data/processed/` - Dados processados e feature engineering
- `notebooks/` - Notebooks de análise exploratória e modelagem
- `src/` - Código fonte (se houver)
- `README.md` - Documentação do projeto
